# Sales Data Analysis Dashboard (Excel Project)

## Project Overview
This project focuses on analyzing sales data and creating an interactive dashboard using Microsoft Excel. The dashboard helps in understanding sales trends, profit performance, and regional sales distribution.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Visualization
- Slicers

## Dashboard Features
- KPI indicators such as Total Sales, Total Profit, Profit Percentage, and Units Sold
- Year-wise Sales Analysis
- Monthly Sales Trends
- Category-wise Sales Performance
- Region-wise Sales Distribution
- Interactive Filters

## Key Insights
- Sales increased from 2018 to 2021
- Consumer segment generated the highest revenue
- West region contributed significant sales
- Technology category performed better compared to others

## Skills Demonstrated
- Data Analysis
- Dashboard Development
- Excel Analytics
- Business Intelligence
